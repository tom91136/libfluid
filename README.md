libfluid
========

Generic fluid simulation solver based on SPH method as described in [Position Based Fluids(ACM TOG 32)](http://mmacklin.com/pbf_sig_preprint.pdf)

## Build dependencies

Build dependencies:

 * cmake
 * clang

Library dependencies:

 * boost-compute
 * OpenMP
 * OpenCL
 * Catch2
 * GLM
 * mio
